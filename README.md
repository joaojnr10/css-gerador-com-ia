Gerador de Código CSS com IA ⚡

- OBS: para funcionar no seu, vc tem que entrar na "https://groq.com/", fazer sua conta, criar "Chave de API"
colocar a chave de acesso no script.js onde esta escrito: "SUA_CHAVE_AQUI"
irá funcionar local, perfeitamente

Este é um projeto web que gera código HTML e CSS automaticamente a partir de uma descrição em texto.
Basta digitar o que você quer (por exemplo: "bola azul quicando") e o sistema cria o código e mostra o resultado em tempo real.

- 🚀 Funcionalidades

Gerar código HTML e CSS com base em texto

Exibir o código gerado

Preview em tempo real do resultado

Interface simples e intuitiva

Layout responsivo

- 🛠️ Tecnologias utilizadas

HTML

CSS

JavaScript

API de IA

- 💻 Como usar

Digite uma descrição na caixa de texto

Clique em Gerar Código

Veja o código gerado

Veja o resultado no preview

Exemplo:

bola azul quicando
🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar JavaScript e integração com APIs, além de demonstrar a criação dinâmica de código HTML e CSS.

- 📌 Observação

Este projeto foi desenvolvido para fins de estudo e portfólio.
